# Anesth-sie-r-animation-
🏥 Référentiel interactif d'anesthésie et de réanimation pour le bloc opératoire. Base de données des drogues majeures et algorithmes de prise en charge des crises vitales (ACR, anaphylaxie, LAST, HM). Développé en HTML5/CSS3/JS pur, responsive et optimisé pour une consultation rapide.
